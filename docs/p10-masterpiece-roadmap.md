## Capstone Acceptance Checklist

A P10 is not done until it has:

- A playable first-session loop of at least 15 minutes.
- Multiple authored levels, chapters, encounters, scenarios, or escalating phases.
- A visual identity distinct from every other course project.
- A serious README with design intent, controls, accessibility notes, build instructions, and known issues.
- A postmortem explaining what the game proves about the course.
- Local and live validation artifacts.
- A Netlify production URL.
- Portfolio metadata and source links updated.
