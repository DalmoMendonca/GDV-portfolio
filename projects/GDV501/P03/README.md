# GDV501-P03: Alchemist's Ledger

Course: GDV 501 - Game Systems, Loops, and Player Experience

Public repository target: games-portfolio/projects/GDV501/P03

## Generic Description

Design resources as verbs; create puzzle constraints from conversion rules; teach through level sequence.

## Specific Execution

Use ore, glass, and signal as resources. Each level shows current stock, exact target stock, and the recipe on every action button so the puzzle teaches conversion, scarcity, and reversibility through play.

## Design Standard

This project is part of the GDV501 First Principles Arcade. It is built as a finished portfolio artifact: playable start and end states, readable controls, deterministic test hooks, responsive canvas presentation, accessibility basics, and Netlify-ready deployment.

## Controls

Click a recipe button. R resets the current ledger.
