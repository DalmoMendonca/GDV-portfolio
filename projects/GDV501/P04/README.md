# GDV501-P04: The Quiet Bid

Course: GDV 501 - Game Systems, Loops, and Player Experience

Public repository target: games-portfolio/projects/GDV501/P04

## Generic Description

Use uncertainty as a system; create readable tells and counterplay; design AI or local multiplayer decision moments.

## Specific Execution

Play five rounds against a readable bot. Probe, feint, or commit based on partial tells. Match recap reveals the hidden values and explains the counterplay.

## Design Standard

This project is part of the GDV501 First Principles Arcade. It is built as a finished portfolio artifact: playable start and end states, readable controls, deterministic test hooks, responsive canvas presentation, accessibility basics, and Netlify-ready deployment.

## Controls

Click Probe, Feint, or Commit

npm install
