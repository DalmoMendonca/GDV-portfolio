# GDV501-P10: Festival Signal

Course: GDV 501 - Game Systems, Loops, and Player Experience

Public repository target: GDV501-P10-festival-signal

## Generic Description

Scope a complete experience for public play; create immediate onboarding; design for spectators and fast turnover.

## Specific Execution

Rotate signal mirrors to route light through three gates before the festival clock expires. The board is readable from a distance and can be completed or restarted in under ten minutes.

## Design Standard

This project is part of the GDV501 First Principles Arcade. It is built as a finished portfolio artifact: playable start and end states, readable controls, deterministic test hooks, responsive canvas presentation, accessibility basics, and Netlify-ready deployment.

## Controls

Click mirrors to rotate, R to restart

## Build Instructions

```bash
npm install
npm run dev
npm run build
```
