# GDV502-P06: Seedfall

Course: GDV 502 - Gameplay Programming and Rapid Tools

Public repository target: games-portfolio/projects/GDV502/P06

## Generic Description

Use randomization responsibly; constrain probability; support replayability without unfairness.

## Specific Execution

A seeded procedural orchard run. Hazards, fruit, wind lanes, and exit placement are generated from a replay code with fairness constraints.

## Grading Rubric

30 pts procedural logic; 20 pts fairness constraints; 20 pts replayability; 15 pts seed/reproduction support; 15 pts notes.

## Controls

Arrows/WASD move, N new seed, C copy seed, R replay

## Build Instructions

```bash
npm install
npm run dev
npm run build
```
