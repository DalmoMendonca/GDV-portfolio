# GDV503-P02: Suitcase Submarine

A tiny submarine where oxygen is shown by cabin bubbles and treasure progress by suitcase stickers mounted in the vessel itself.

## Learning Objective
Integrate UI into the game world; reduce HUD dependency; communicate health, score, or objectives through objects and environment.

## Controls
Arrows/WASD pilot, Space ping

## Assessment Rubric
30 pts diegetic UI clarity; 20 pts world integration; 20 pts gameplay completeness; 15 pts visual hierarchy; 15 pts playtest evidence.

## Build
```bash
npm install
npm run build
npm run dev
```
