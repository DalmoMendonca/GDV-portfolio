## Generic Description

Integrate UI into the game world; reduce HUD dependency; communicate health, score, or objectives through objects and environment.

## Specific Execution

Build a game where at least two essential status indicators are diegetic. Players should not need a conventional corner HUD to understand progress or danger.

## Grading Rubric

30 pts diegetic UI clarity; 20 pts world integration; 20 pts gameplay completeness; 15 pts visual hierarchy; 15 pts playtest evidence.
