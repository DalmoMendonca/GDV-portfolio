# GDV503-P03: Chime Orchard

A listening garden where hidden chimes get louder and pulse visually as the player approaches, with captions and a no-audio pulse trail.

## Learning Objective
Design gameplay around listening; mix audio cues by priority; provide non-audio alternatives for accessibility.

## Controls
Arrows/WASD move, Space sonar, C captions

## Assessment Rubric
30 pts audio mechanic design; 20 pts mix clarity; 20 pts accessible alternatives; 15 pts complete game loop; 15 pts documentation.

## Build
```bash
npm install
npm run build
npm run dev
```
