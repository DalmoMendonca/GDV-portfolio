# GDV503-P08: Heartbeat Desk

A pressure-management desk game whose interface breathes, tightens, and relaxes with player condition while preserving legible controls.

## Learning Objective
Connect UI state to emotion; use typography, pacing, sound, and layout to shape feeling; maintain usability under expressive presentation.

## Controls
Arrows/WASD move, Space file task, B breathe

## Assessment Rubric
30 pts expressive UI concept; 20 pts usability; 20 pts technical implementation; 15 pts emotional coherence; 15 pts reflection.

## Build
```bash
npm install
npm run build
npm run dev
```
