# GDV503-P09: Access Kitchen

A cooking microgame with live toggles for captions, high contrast, slow mode, larger text, simplified motion, remap hinting, and difficulty assist.

## Learning Objective
Audit barriers; design options for diverse players; evaluate whether an accessibility feature changes the core challenge appropriately.

## Controls
Arrows/WASD move, Space serve, 1-6 accessibility toggles

## Assessment Rubric
30 pts quality of accessibility improvements; 20 pts preservation of core design; 20 pts implementation completeness; 15 pts before/after evidence; 15 pts audit rationale.

## Build
```bash
npm install
npm run build
npm run dev
```
