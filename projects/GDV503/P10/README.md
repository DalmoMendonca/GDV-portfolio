Combine game feel, UI, audio, and presentation; design for public play; present a coherent brand identity.

## Specific Execution

Build a polished arcade-style game with title treatment, attract loop, tutorial prompt, sound mix, score feedback, and end screen. It must be understandable from a standing demo station.
