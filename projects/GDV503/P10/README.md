# GDV503-P10: Poppy Cabinet

A stand-up micro-arcade cabinet with attract loop, title treatment, onboarding prompt, score burst, end screen, and public-play pacing.

## Learning Objective
Combine game feel, UI, audio, and presentation; design for public play; present a coherent brand identity.

## Controls
Arrows/WASD move, Space pop, R restart

## Assessment Rubric
30 pts complete audiovisual polish; 20 pts onboarding; 20 pts game feel; 15 pts public-play readiness; 15 pts documentation.

## Build
```bash
npm install
npm run build
npm run dev
```
