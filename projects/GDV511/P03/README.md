# GDV511-P03: Compass Plaza

A four-spoke hub where each route teaches a different spatial problem and returns with a landmark reward that changes the central plaza.

## Learning Objective
Design navigable hubs; create landmarks and return paths; reward optional exploration.

## Controls
Arrows/WASD move, Space interact, R restart

## Assessment Rubric
30 pts hub navigation; 20 pts spoke variety; 20 pts reward design; 15 pts landmark readability; 15 pts documentation.

## Build
```bash
npm install
npm run build
npm run dev
```
