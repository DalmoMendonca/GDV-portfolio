## Generic Description

Design navigable hubs; create landmarks and return paths; reward optional exploration.

## Specific Execution

Build a small hub with at least four spokes. Each spoke must contain a distinct challenge and a reward or information piece that changes how the hub is perceived or used.

## Grading Rubric

30 pts hub navigation; 20 pts spoke variety; 20 pts reward design; 15 pts landmark readability; 15 pts documentation.
