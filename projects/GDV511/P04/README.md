# GDV511-P04: Moonlit Orchard Outpost

A gentle stealth outpost with three patrols, two orchard routes, recoverable suspicion, grass cover, and a clear objective line.

## Learning Objective
Design patrol routes, vision cones, cover, and escape paths; balance failure states; create readable stealth affordances.

## Controls
Arrows/WASD sneak, Space crouch, R restart

## Assessment Rubric
30 pts stealth encounter design; 20 pts route choice; 20 pts feedback and readability; 15 pts recoverability; 15 pts notes.

## Build
```bash
npm install
npm run build
npm run dev
```
