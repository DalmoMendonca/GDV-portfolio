# GDV511-P05: Clockwork Courtyard

A three-phase encounter where gates, cover, enemy pressure, and resources rotate through the arena without hiding the rules.

## Learning Objective
Shape combat through space; use enemy composition and arena changes; pace intensity across phases.

## Controls
Arrows/WASD move, Space strike, R restart

## Assessment Rubric
30 pts encounter pacing; 20 pts arena affordances; 20 pts phase differentiation; 15 pts fairness and telegraphing; 15 pts testing.

## Build
```bash
npm install
npm run build
npm run dev
```
