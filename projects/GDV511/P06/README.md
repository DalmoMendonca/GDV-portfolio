# GDV511-P06: The Last Greenhouse

An explorable greenhouse that tells a recovery story through broken paths, staged props, growth pockets, and discovered object relationships.

## Learning Objective
Communicate narrative through arrangement; use props and spatial sequence; avoid exposition as the main delivery method.

## Controls
Arrows/WASD move, Space inspect, R restart

## Assessment Rubric
30 pts environmental storytelling; 20 pts spatial sequencing; 20 pts interaction design; 15 pts interpretive clarity; 15 pts reflection.

## Build
```bash
npm install
npm run build
npm run dev
```
