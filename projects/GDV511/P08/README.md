# GDV511-P08: Five Rooms One Rule

Five compact levels built around a single dash-to-token mechanic, each adding a new constraint: space, timing, enemy, information, and resource.

## Learning Objective
Reuse a mechanic across multiple contexts; create variety through constraints; stage difficulty progression.

## Controls
Arrows/WASD move, Space dash, R restart

## Assessment Rubric
30 pts constraint design; 20 pts level variety; 20 pts progression; 15 pts clarity; 15 pts playtest evidence.

## Build
```bash
npm install
npm run build
npm run dev
```
