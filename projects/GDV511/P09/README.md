# GDV511-P09: Harbor Errands

A nonlinear harbor mission with three objectives, route-changing dependencies, and visible state changes that make planning matter.

## Learning Objective
Design objectives that can be completed in multiple orders; manage dependency and player planning; support meaningful route choice.

## Controls
Arrows/WASD move, Space complete objective, R restart

## Assessment Rubric
30 pts nonlinear structure; 20 pts dependency design; 20 pts route choice; 15 pts state communication; 15 pts documentation.

## Build
```bash
npm install
npm run build
npm run dev
```
