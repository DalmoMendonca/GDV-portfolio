Use materials, render passes, depth, normals, masks, or engine-equivalent techniques to communicate affordances; separate visual effect from game logic.

## Specific Execution

Build a game where interactable, dangerous, hidden, or selected objects are communicated through a custom outline, highlight, x-ray, silhouette, or screen-space effect. Include at least three gameplay states with distinct visual treatment.
