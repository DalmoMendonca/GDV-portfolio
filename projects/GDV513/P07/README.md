## Generic Description

Represent place through interaction; use spatial design to express social, cultural, aesthetic, or computational meaning; avoid generic backdrops.

## Specific Execution

Build a game set in a specific place, real or imagined. The place must shape mechanics, not only visual theme. Include a short context note or artist statement.

## Grading Rubric

30 pts place-mechanic connection; 20 pts specificity; 20 pts level/spatial quality; 15 pts track interpretation; 15 pts documentation.
