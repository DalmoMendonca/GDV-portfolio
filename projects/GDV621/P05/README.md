## Generic Description

Track player behavior; update NPC disposition; model short-term and long-term memory; communicate relationship state through action.

## Specific Execution

Build a game where at least two NPCs remember player actions and change available interactions, help, resistance, dialogue, or strategy. At least one memory must persist across scenes or sessions. Avoid purely cosmetic affinity meters.

## Grading Rubric

30 pts relationship/memory model; 20 pts consequential NPC behavior; 20 pts clarity to player; 15 pts state persistence; 15 pts documentation.
