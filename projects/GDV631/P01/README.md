## Generic Description

Define an audience segment; build for a tested need or desire; compare assumptions to evidence.

## Specific Execution

Build a game for a clearly defined audience. Before building, write three audience hypotheses; after testing, report which were supported, revised, or rejected.

## Grading Rubric

30 pts audience fit; 20 pts hypothesis quality; 20 pts playtest evidence; 15 pts game completeness; 15 pts reflection.
