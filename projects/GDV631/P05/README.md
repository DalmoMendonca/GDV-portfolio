## Generic Description

Build toward learning, training, health, civic, or organizational outcomes; connect outcome to assessment; avoid reducing the game to a quiz.

## Specific Execution

Build an applied game with a measurable non-entertainment objective. Include a logic model explaining inputs, activities, outputs, and desired outcomes.

30 pts applied design quality; 20 pts outcome alignment; 20 pts assessment method; 15 pts playability; 15 pts documentation.
