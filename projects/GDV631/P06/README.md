# GDV631-P06: User Research Instrument Game

Course: GDV 631 - Business, Ethics, Accessibility, and Player Research

Netlify status: Placeholder. Deploy this game when implementation begins.

## Generic Description

Design a game to answer a research question; build observation, survey, and A/B testing tools; separate findings from impressions.

## Specific Execution

Build a game and run a small study with at least five participants. The study must include either an A/B-tested tutorial, difficulty variant, UI variant, or onboarding variant. Submit the game, protocol, consent script, observation sheet, segmentation method, and findings summary.

## Grading Rubric

25 pts game suitability; 25 pts research protocol and segmentation; 20 pts findings quality; 15 pts ethical handling; 15 pts reflection.
